## 索引类型

* 从存储引擎分析

  > Innodb：聚簇索引
  >
  > MyISAM：非聚簇索引（辅助索引）
  >
  > * 聚簇索引：B+树叶子节点存储数据而不是主键
  > * 非聚簇索引（辅助索引）：B+树叶子节点存储的是主键

* 类型：

  - 主键索引
  - 普通索引
  - 唯一索引
  - 联合索引
  - 全文索引

## 索引分析

* 索引区分度

  > show index from table 查看所有区分度
  >
  > 索引区分度越大越好，通过此索引过滤掉的数据越多

* explain语法分析

  * Type
    * system ：表中只有一行数据
    * const：一般是主键索引或者唯一索引
    * eq_ref：唯一性索引扫描
    * ref：非唯一性索引扫描
    * range：范围查找
    * index：只查找索引即可获取数据，涉及索引覆盖
    * all：全表扫描
  * extra
  * rows
