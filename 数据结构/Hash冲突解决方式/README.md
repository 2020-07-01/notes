## 再哈希法

>

## 开放地址法(推荐)

公式：hashCode(i) = (hash(key) + di) % m    (di = 1，2，3，，，n；m为表长)

> 当出现哈希冲突时，再次计算hashCode，直到不冲突为止

m为表长，di为增量序列，di得取值方式不同，再散列方式也不同，可分为三种

* 线程探测再散列

  > di 线性增长

* 二次探测再散列

  > 在冲突点的左右两侧跳跃探测  

* 随机探测法

  > 取随机值
